## O'rnatish

1. `UstajonSupport.exe` faylini yuklab oling
2. Ishga tushiring
3. Ismingiz va telefon raqamingizni kiriting
4. "Yordam olish" tugmasini bosing

2. Sizning ma'lumotlaringiz serverga yuboriladi
